QvPlayer 隐私政策

最近更新日期：2026年1月1日

QvPlayer（以下简称“本应用”）非常重视您的隐私。本政策旨在说明我们如何处理您的信息。

1. 数据收集

我们不收集任何个人身份信息。 QvPlayer 是一款本地播放器工具，所有的播放列表、视频地址和配置信息均存储在您的设备本地（Apple TV）。

2. 局域网访问

本应用请求局域网访问权限仅用于以下目的：

  • 提供内置的 Web 管理界面，方便您从电脑浏览器管理播放列表。
  • 发现并播放您局域网内的媒体资源。

3. 第三方服务

本应用集成了 KSPlayer 和 FFmpeg 等开源库用于视频解码。这些库在本地运行，不会向第三方发送您的数据。

4. 数据安全

由于所有数据均存储在本地，数据的安全性取决于您的设备安全。我们建议您保护好您的局域网环境。

5. 联系我们

如果您对本隐私政策有任何疑问，请通过应用内的支持页面联系我们。
