常见问题 (FAQ)

  • 如何导入视频？ 您可以在设置中开启 Web 管理界面，然后在电脑浏览器访问显示的 IP 地址进行导入。
  • 视频没有画面只有声音？ 请在设置中将“播放引擎”切换为 KSPlayer (FFmpeg)。
  • 无法连接 Web 界面？ 请确保您的 Apple TV 和电脑处于同一个局域网（Wi-Fi）下。
