QvPlayer Support

Thank you for using QvPlayer! If you encounter any issues during use, you can get help through the following ways.

Frequently Asked Questions (FAQ)

  • How to import videos? You can enable the Web Management Interface in Settings, then access the displayed IP address from a computer browser to import.
  • Video has sound but no image? Please switch the "Player Engine" to KSPlayer (FFmpeg) in Settings.
  • Cannot connect to the Web Interface? Please ensure your Apple TV and computer are on the same local network (Wi-Fi).
